ТЕЛЕФОННЫЙ СПРАВОЧНИК

Функции:
-Постраничное чтение содержимого
-Добавление новых записей в справочник
-Редактирование существующих записей
-Поиск записей по id, а также по характерикам (одной или несколько)

Классы:
-Справочник (Phonebook)
-Запись (Record)

Поля и методы классов:
-Phonebook:
    +Поля:
        *records (Список элементов пользоватеьского типа Record)
        *page_count (Количество страниц в справочнике)
    +Методы:
        *add_record (Метод, добавляющий запись в конец справочника)
        *read_pb (Выводит в консоль содержимое страницы справочника)
        *find_records (Метод линейного поиска записей в справочнике по заданной характеристике, возвращает результаты в виде множества)
        *find_records_by_multiple (Линейный поиск по нескольким характеристикам, использует find_records)
        *edit_record (Изменяет значение характеристики выбранной по id записи)
        *__save_pb (Приватный метод, отвечающий за обновление и сохранение справочника в файл)

-Record:
    +Поля:
        *d (Словарь содержащий все данные о записи)
    +Методы:
        *get_info (Возвращает строку с информацией о записи)
        *get_row (Возвращает строку для записи в текстовом файле)

Запуск:
    Запустите через консоль или IDE main.py
